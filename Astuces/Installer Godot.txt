1 dépendances :
    Requis (pour la version dotnet) :
    SDK Dotnet





    Pour installer dotnet via un gestionnaire de paquet :
    Chercher la dernière version du SDK avec la commande
apt search dotnet-sdk

    ensuite l'installer :
sudo nala install -y dotnet-sdk-7.0


Pour vérifier l'installation :
dotnet --list-sdks
ou
dotnet --info

----------------------------------------------------

2 Installer Godot :

flatpak install -y flathub org.godotengine.GodotSharp


ou


Allez sur la page :
https://godotengine.org/download/linux/

et cliquez sur "Godot Engine -.NET" pour lancer le téléchargement.
Je recommande de mettre l'archive dans ~/Local/Godot/

Dans la commande suivante, remplacez "$Home" par "/home/*votreNom*"
et vérifiez que le fichier s'appelle bien "Godot_v4.1.3-stable_mono_linux.x86_64"

echo -e '[Desktop Entry]\nVersion=1.0\nType=Application\nName=Godot Engine (C#)\nIcon=godot\nExec=$HOME/Local/Godot/Godot_v4.1.3-stable_mono_linux.x86_64 %u\nType=Application\nMimeType=application/x-godot-project;\nCategories=Development;IDE;;' | sudo tee /usr/share/applications/godot.desktop
