Pour connaître la taille du fichier de swap de lm :
swapon
# Par défaut c'est 2Go

Dans l'exemple je vais mettre 16Go pour le swap, ce qui fait la valeur 16384
1 Go = 1024

Arrêter le swap utilisé par le système :

sudo swapoff -a

Créer le fichier swap et lui donner les permissions (count c'est pour la taille) : 

sudo dd if=/dev/zero of=/swapfile bs=1M count=16384
#attendre un peu, le fichier est en court de création
sudo chmod 0600 /swapfile
sudo mkswap /swapfile

Activer le fichier swap

sudo swapon -a
